TXT Reviews

TXT Reviews is a customer feedback tool for any business that has regular customer interactions. It is perfect for restaraunts, retail, and other service industries. Reviews and feedback are solicited via text messages sent to a location specific phone number. The goal is to provide a minimally frictional way for the customer to give realtime feedback to the business.

Google Books

React-based Google Books Search app. It is made with React components, helper/util functions, and utilize React lifecycle methods to query and display books based on user searches. It uses Node, Express and MongoDB so that users can save books to review or purchase later.

Clicky Game

A memory game with React. It is broken up by application's UI into components, manage component state, and respond to user events. The application renders different images to the screen. Each image listens for click events.

Mongo Scraper

A web app that lets users view and leave comments on the latest news that are being scraped from another site using Node.js with Mongoose and Cheerio. It also displays clothes/dresses and let's user saved each item.

Dasher

A group project that gathers your selected information from across the web and displays it in one convenient location. It uses Javascript, Node, Express, MySQL, Handlebars, Sequelize, and Bootstrap Git for version control and deployed in Heroku.

Eat-Da-Burger!

A burger logger application with Javascript, Node, Express, Handlebars, ORM/Sequelize, MySQL, and Bootstrap deployed in Heroku.

Friend Finder

It is like a dating app using Javascript, Node.js based application with Express, and Bootstrap deployed in Heroku.
